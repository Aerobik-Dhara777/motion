## 🎯 Getting Started (3 Steps)

### 1. Extract & Install
```bash
tar -xzf luxe-travel-agency.tar.gz
cd v0-project
pnpm install
```

### 2. Run Locally
```bash
pnpm dev
# Open http://localhost:3000
```

### 3. Deploy
```bash
# Vercel (recommended)
vercel

# Or build & deploy anywhere
pnpm build
pnpm start
```
